Worum geht es?

„3D Connect Four“ ist ein reines Offline-Spiel. Es benötigt keine Registrierung und sendet keine personenbezogenen Daten an mich oder Dritte.

Welche Daten verarbeiten wir?

  • Keine Übermittlung personenbezogener Daten an Server.
  • Lokaler Speicher auf deinem Gerät. Das Spiel speichert z. B. Spielstände/Zähler ausschließlich lokal (z. B. in localStorage/sessionStorage oder im Cache für Offline-Nutzung). Diese Daten verlassen dein Gerät nicht.
  • Keine Analyse-Tools, keine Werbung, keine In-App-Käufe.

Hosting/Logs

Die Website wird statisch über GitHub Pages ausgeliefert. Dabei kann GitHub zur Bereitstellung der Seiten technisch notwendige Protokolldaten (z. B. IP-Adresse, Zeitpunkt, User-Agent) verarbeiten. Weitere Infos findest du in den Datenschutzhinweisen von GitHub.

Berechtigungen

Die App/Website fordert keine Geräterechtigungen (z. B. Kamera, Standort) an.

Löschen lokaler Daten

Du kannst die lokal gespeicherten Daten jederzeit löschen: Im Browser über Website-Daten/Cache für diese Domain. In der Android-App (TWA) über Systemeinstellungen → Apps → Daten/Cache löschen oder die App deinstallieren.

Kinder

Die App richtet sich nicht speziell an Kinder unter 13 Jahren. Es werden keine Daten von Kindern wissentlich erhoben. Wenn du glaubst, dass dennoch Daten erfasst wurden, kontaktiere mich bitte – ich lösche sie umgehend (sofern vorhanden).

Änderungen
